新疆大学飞跃手册

为新疆大学学生打造的开放知识社区 — 笔记、资料、选校、学分，一站直达

STATUS · LIVE IN PRODUCTION DATE · 2026-06-04 LIVE · winbeau.top MIRROR · feiyue.selab.top
TL;DR
新疆大学飞跃手册是一个面向 XJU 在校生的全栈知识平台，覆盖科研笔记、课程资料、选校情报、CCF 会议截止日期与学分达标自检，于 2026-05-09 单日内完成前后端基础架构（由 Claude Opus 4.7 完成 5 轮结构化设计），此后历经 26 天迭代上线十大功能模块。截至 2026-06-04，已有 108 位注册用户，累积 85 篇笔记、55 个共享文件、41 个点赞，25 名用户活跃登录，项目代码托管于 winbeau.top。

01项目简介

飞跃手册起源于一个简单的需求：让 XJU 学生在申请科研、选择高校、备战竞赛时，能找到来自真实同学、而非碎片化论坛的第一手经验。平台以 Markdown 长文笔记为核心，支持 KaTeX 公式、围栏代码块、GFM 表格与评论互动；配套的 资料库（/materials）可上传、预览、分类管理课程文件；高校导师库（/schools）和 CCF 会议页（/conferences）打通选校与科研信息流；学分统计页（/credits）更可直接解析教务成绩单 PDF，自动核查通识选修模块达标情况。

线上地址：https://winbeau.top（主站）  /  https://feiyue.selab.top（团队镜像）

Markdown 笔记 KaTeX 公式 AI 润色（DeepSeek） 资料上传/预览 高校/导师库 CCF 会议截止日期 学分达标检查 教务一键导入 点赞 / 评论 个性化欢迎语 草稿自动保存 全文检索

代表性 Bug 攻坚

26 天迭代中，几处 bug 尤为棘手，留下了深刻的工程经验：

Bug 根因 状态
成绩单 PDF 列错位 pdf.js 以表头标签 x 坐标划分列边界，但课程名数据左对齐（x≈88）而表头居中（x≈160），导致短课程名归入序号列，整行记录串行。修复：扫描实际课程代码最小 x 值重新划定分界点。 FIXED
tmux 双跳 SSH 静默损坏 经 WSL→huawei2 双层 pty send-keys 传输 ~1.1 KB base64 长行时，pty 行缓冲翻转 1 字节，head/tail 抽样无法发现。修复：分块传输（每块 ≤300 字符）+ sha256 闸门 + 规范化哈希反验。 FIXED
WebVPN Referer 污染 CAS window.open(url, '_blank') 把 feiyue.selab.top 带入 Referer，WebVPN 将其拼进 CAS 回调导致登录失败。修复：改用 noreferrer 阻止 Referer 发送。 FIXED
sonner toast 不渲染 sonner v2 + React 18 StrictMode + Vite HMR + Radix Dialog teardown 组合下，Toaster 的 subscribe-effect 存在 stale gap，Dialog 触发的 toast 恰好落在 gap 里被丢弃。修复：回退到 <Toaster position="bottom-right" richColors /> 基线，不叠加 classNames 重写。 FIXED
nginx 路径歧义 /conferences 同时是 SPA 前端路由和后端 API 路径，try_files 优先返回 index.html，前端 useQuery 拿到 HTML 字符串后 silently fallback 到旧 seed 数据。修复：API 路径改为 /conferences/list，与 SPA 路由不再冲突。 FIXED

完整 bug 记录见项目 CONTRIBUTING.md 与各 feature memory 文件。
